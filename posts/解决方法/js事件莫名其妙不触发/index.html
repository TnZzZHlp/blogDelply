JS事件莫名其妙不触发解决方法

June 16, 2025

问题描述#

监听 JavaScript 事件时，对应处理函数只有一个console.log()，但事件并没有触发。

document.getElementById("myButton").addEventListener("click", (e) => {
    console.log("Button clicked");
});

问题原因#

Vite 中配置了去除 console.log 的功能，导致在生产环境中无法输出日志。

解决方法#

删除对应的 Vite 配置即可
